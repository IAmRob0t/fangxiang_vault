# Prompt：Markdown 笔记深度重构与原子化优化指令

## 1. 核心目标

请将一篇现有的、结构可能较为松散的 Markdown 笔记及其引用的图片，进行一次**深度的重构和优化**。最终目标是产出一篇**逻辑清晰、结构严谨、图文并茂、内容简洁易懂**、且遵循“文件名即标题”（原子化笔记）原则的终稿。

---

## 2. 上下文与文件信息

在执行任务前，请确认并提供以下路径信息：

- **笔记文件路径**: `00_Inbox/06_数据处理指令.md`
- **图片目录路径**: `00_Inbox/attachments/06_数据处理指令`

---

## 3. 核心任务与流程 (三阶段精细化执行)

这是一个多步骤的、精细化的重构任务。请**严格**按照以下流程执行，并仔细检查每一步的正确性。

### **第一阶段：分析、提炼与规划**

1. **全面分析与提炼**：
    - 仔细阅读笔记的全部内容，**提炼**其核心主题、关键概念和当前信息组织方式。
    - 逐一查看图片目录下的所有图片，理解每张图片所展示的具体内容、操作或意图。
2. **制定重构大纲与命名规范**：
    - 根据笔记的核心主题，设计一个全新的、逻辑层层递进的**内容大纲**。推荐使用“是什么（概念）- 为什么（原理）- 怎么做（步骤/实操）”的结构。
    - 为每一张图片规划一个**清晰、准确、具有描述性的新文件名**。文件名必须使用**英文或中文无空格**连接（例如：`VisUAL主界面概览.png` 或 `VisUAL_main_interface.png`）。

### **第二阶段：执行重命名与内容重写 (强调易读性)**

3. **重命名图片文件**：
    - 根据第2步的规划，在文件系统上**实际地**将图片文件从旧名称**重命名**为新名称。请务必确保此操作已完成。
4. **笔记内容重写与图文整合**：
    - 基于新的内容大纲，将原有笔记的零散信息进行整合、提炼和**彻底重写**。
    - **重点要求：** 使文本**简洁易懂**，逻辑通顺，**用最简单的语言解释复杂的概念**，确保**初学者**也能无障碍地读懂并理解内容。
    - 在重写过程中，将已重命名的图片以 Markdown 链接的格式 `![](图片路径/新文件名.png)` **准确**地插入到其描述内容的最恰当位置。

### **第三阶段：最终结构化与原子化命名 (关键)**

5. **识别“终极原子标题”**：
    - 从重构后的笔记内容中，确定最顶层的、概括了整篇笔记的**核心标题**（例如：《VisUAL模拟器入门教程》），题目**不能带有空格**。
6. **内容格式化：剥离标题与层级提升**：
    - **删除**笔记正文中的那个顶层一级标题（即 `# 标题`）。（原子化笔记的标题由文件名承载）。
    - 将笔记中所有剩余的标题**提升一个层级**。例如，原二级标题 (`##`) 变为新的一级标题 (`#`)，三级标题 (`###`) 变为新的二级标题 (`##`)，以此类推。
7. **文件原子化命名**：
    - 最后，将这个 `.md` 笔记文件本身**重命名**为第5步中识别出的“终极原子标题”。（例如：`06_数据处理指令.md` 重命名为 `VisUAL模拟器入门教程.md`）。

---

## 4. 🚀 最终交付成果检查清单

- **[文件]** 一个经过重命名的 `.md` 文件（文件名即标题）。
- **[结构]** 文件内容内部无重复的最高级标题。
- **[内容]** 所有标题层级已正确提升一级。
- **[可读性]** 文本内容简洁易懂，逻辑层层递进，适合初学者阅读。
- **[附件]** 一个存放着所有**已重命名**图片的附件文件夹。
- **[链接]** 笔记中所有图片链接均已更新为新的文件名。
