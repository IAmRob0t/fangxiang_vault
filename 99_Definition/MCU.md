# 上拉输入

*Pull-up Input*

- **工作原理**：内部上拉电阻（约20-50kΩ）将引脚默认拉至高电平。当外部信号拉低时，引脚电平变低。
- **典型应用**：按键检测（按键接地）、默认需高电平的输入信号。
- **示例**：按键一端接地，按下时引脚被拉低，释放时内部上拉至高电平。

---

# 下拉输入

*Pull-down Input*

- **工作原理**：内部下拉电阻将引脚默认拉至低电平。当外部信号拉高时，引脚电平变高。
- **典型应用**：按键接电源、默认需低电平的输入信号。
- **示例**：按键一端接VCC，按下时引脚被拉高，释放时内部下拉至低电平。

---

# 浮空输入

*Floating Input*

- **工作原理**：输入引脚内部无上拉或下拉电阻，电平完全由外部电路决定。若外部未连接，电平可能不稳定（易受噪声干扰）。
- **典型应用**：需要外部上拉/下拉的场景，如I2C总线（需外接上拉电阻）、数字传感器信号线。
- **注意事项**：需确保外部电路提供确定电平，否则可能读取到随机值。

---

# 模拟输入

*Analog Input*

- **工作原理**：禁用数字输入电路（施密特触发器、上拉/下拉电阻），信号直接接入ADC或比较器。
- **典型应用**：连接模拟传感器（如温度传感器）、ADC采样、低噪声模拟信号。
- **注意事项**：此模式下无法读取数字信号（输入数据寄存器无效）。

---

# 推挽输出

*Push-Pull Output*

- **工作原理**：可主动输出高（PMOS导通）或低（NMOS导通）电平，驱动能力强。
- **典型应用**：
    - 驱动LED、继电器等需高电流设备。
    - 高速数字信号（如SPI、USART）。
- **优点**：高低电平切换快，抗干扰能力强。

---

# 开漏输出

*Open-Drain Output*

- **工作原理**：仅能主动拉低电平（NMOS导通），高电平需依赖外部上拉电阻。支持“线与”逻辑。
- **典型应用**：
    - I2C总线（多设备共享总线）。
    - 电平转换（如3.3V MCU与5V设备通信）。
- **优点**：避免总线冲突，兼容不同电压设备。
- **缺点**：上升时间依赖外部上拉电阻，高速场景需小电阻。

---

# 复用推挽输出

*Alternate Function Push-Pull*

- **工作原理**：外设（如SPI、USART）控制推挽输出，驱动能力与普通推挽相同。
- **典型应用**：
    - SPI的MOSI/MISO、USART的TX引脚。
    - 定时器PWM输出（驱动电机、LED）。
- **配置要点**：需正确映射外设到指定GPIO引脚。

---

# 复用开漏输出

*Alternate Function Open-Drain*

- **工作原理**：外设（如I2C）控制开漏输出，其余特性与普通开漏相同。
- **典型应用**：I2C的SDA/SCL引脚、定时器PWM输出（需特殊场景）。 
- **配置要点**：需通过AFR（Alternate Function Register）选择外设功能。

---

# I2C

I²C（Inter-Integrated Circuit）集成电路总线。I²C是一种串行通信总线，采用多主从架构，由飞利浦公司在1980年代开发，用于连接主板、嵌入式系统或手机等低速外围设备。

---

# EXTI

*外部中断/事件控制器（External interrupt/event controller）*

EXTI（外部中断/事件控制器）是嵌入式系统（如 STM32 微控制器）中用于实时检测和响应外部信号变化（如 GPIO 引脚电平跳变），并通过触发中断或事件机制通知处理器执行相应操作的核心外设模块。

---

# NVIC

*嵌套向量中断控制器（Nested vectored interrupt controller）*

NVIC（嵌套向量中断控制器）是 ARM Cortex-M 系列处理器中用于高效管理和优先处理中断的模块，通过嵌套中断和向量化机制实现快速、实时的中断响应。

---
# GPIO

*通用输入/输出（General-purpose input/output）*

通用输入/输出( GPIO ) 是集成电路或电子电路（例如MCU / MPU ）板上未指定的数字信号引脚，可用作输入或输出，或两者兼而有之，并可由软件控制。

---
# 上升沿触发

*rising edge trigger*

在信号从低电平跳变到高电平的瞬间触发相应操作。

---
# 下降沿触发

*falling edge trigger*

在信号从高电平跳变到低电平的瞬间触发相应操作。

---
# UART

*通用异步收发器（Universal Asynchronous Receiver Transmitter）*

UART（Universal Asynchronous Receiver/Transmitter）是一种采用异步串行通信协议的硬件设备或电路，通过**起始位、数据位、可选的校验位和停止位**构成数据帧，在发送端将并行数据转为串行信号传输，接收端则逆向还原数据，依靠预定义的**波特率**实现设备间时序同步，无需共享时钟信号即可完成全双工通信，广泛应用于微控制器、传感器、传统外设等低速短距离数据交互场景。

---
# 比特率


每秒传输的二进制位数（bps），反映有效信息传输速度。  

---
# 波特率

每秒传输的符号数（Baud），表征信号波形变化速率。  

---
# 全双工

全双工是通信技术中的一种数据传输模式，允许双方在同一时间内既能发送又能接收信息，实现双向实时交互（如电话通话、视频会议），其核心在于通过独立的传输通道或技术手段（如频分复用）消除信号冲突，从而区别于同一时间只能单向传输的半双工（如对讲机）和仅单向传输的单工模式（如广播）。该技术广泛应用于现代网络通信、无线系统等领域，支撑高效同步的数据交换。
