# Prompt：Markdown 笔记深度重构与优化指令

## 1. 核心目标

请将一篇现有的、结构可能较为松散的 Markdown 笔记及其引用的图片，进行一次深度的重构和优化。最终目标是产出一篇逻辑清晰、结构严谨、图文并茂、且遵循“文件名即标题”原则的原子化笔记。

---

## 2. 上下文信息

在执行任务前，请提供以下信息：

*   **笔记文件路径**: `[请在这里填入需要重构的 .md 文件路径]`
*   **图片目录路径**: `[请在这里填入存放相关图片的目录路径]`

---

## 3. 核心任务与流程

这是一个多步骤的、精细化的重构任务。请严格按照以下流程执行，并仔细检查每一步的正确性。

### **第一阶段：分析与规划**

1.  **全面分析**：
    *   仔细阅读笔记的全部内容，理解其核心主题和当前的信息组织方式。
    *   逐一查看图片目录下的所有图片，理解每张图片所展示的具体内容、操作或意图。

2.  **制定重构计划**：
    *   根据笔记的核心主题，设计一个全新的、逻辑层层递进的**内容大纲**（例如，从“是什么”到“为什么”再到“怎么做”，或者按照教程的步骤顺序）。
    *   为每一张图片规划一个清晰、准确、具有描述性的**新文件名**（例如，将 `IMG_1234.png` 规划为 `VisUAL主界面概览.png`）。

### **第二阶段：执行重构**

3.  **重命名图片文件**：
    *   根据上一步的规划，在文件系统上**实际地**将图片文件从旧名称重命名为新名称。这是关键一步，请确保操作的是真实文件。

4.  **重写笔记内容**：
    *   基于新的内容大纲，将原有笔记的零散信息进行整合、提炼和重写，使其语言更流畅、结构更清晰。
    *   在重写过程中，将已重命名的图片以 Markdown 链接的格式 `![图片描述](图片路径/新文件名.png)` 插入到最恰当的位置。

### **第三阶段：最终结构化与命名（关键步骤）**

5.  **识别笔记的“真正标题”**：
    *   从重构后的笔记内容中，确定最顶层的核心标题（通常是原笔记的第一个一级标题）。

6.  **从内容中剥离标题**：
    *   **删除**笔记正文中的那个顶层一级标题（`# 标题`）。笔记的“标题”将由文件名来承载，正文内部不应再有重复的最高级标题。

7.  **提升所有标题层级**：
    *   将笔记中所有剩余的标题**提升一级**。例如，二级标题（`##`）变为新的一级标题（`#`），三级标题（`###`）变为新的二级标题（`##`），以此类推。

8.  **重命名笔记文件**：
    *   最后，将这个 `.md` 笔记文件本身**重命名**为第5步中识别出的“真正标题”。（例如，将 `草稿笔记.md` 重命名为 `VisUAL模拟器入门教程.md`）。

---

## 4. 最终交付成果

*   一个经过重命名的、内容和结构都已焕然一新的 `.md` 文件。
*   一个存放着所有已重命名图片的附件文件夹。
*   整个目录结构干净、清晰，符合原子化笔记的最佳实践。
