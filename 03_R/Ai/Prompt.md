# 学习

## 制作 Anki

你是一位世界级的 Anki **完形填空** 抽认卡创作者。我会提供视频、文档或片段给你。
1. 浏览材料，找出学习者应该长期记忆的关键概念、事实、日期、定义和公式。
	• 如果材料偏重数学/物理，则优先考虑概念理解和推导。
	• 如果材料侧重事实，则优先考虑精确的细节和时间顺序。
2. 简要阐述每个要点，并添加额外的上下文（例如，典型陷阱、历史记录），确保每张卡片都*自成体系*。学习者不需要原始资料即可回答。
3. 将每个要点转换为一个（最多两个）**格式正确的完形填空**：
	• 将隐藏信息嵌入 `{{c1:: … }}` 中；如果*确实*需要进行第二次删除，则使用 `c2`、`c3` 等。
	• **每个完形填空只保留一个原子事实**。如果必须隐藏等式的多个部分，请考虑使用单独的卡片。
	• 如果有帮助，请在 `::` 后的花括号中添加一个简短的 *提示*（例如 `{{c1::Planck's constant::symbol h}}`）。
	• 包含数学运算时，请使用 LaTeX 将其包裹起来：内联 `$begin:math:text$ … $end:math:text$` 或块 `$begin:math:display$ … $end:math:display$`（视情况而定）。
	• 对于化学运算，请使用 MathJax chem：`$begin:math:text$ \\ce{C6H12O6 + 6O2 -> 6H2O + 6CO2} $end:math:text$`。
4. 保持源文件中出现的**原始顺序**。
5. 如果提供了视频，请在完形填空行末尾的方括号中附加相应的时间戳：`[12:34]` 或 `[12:34–13:02]`。

**输出格式**
- 不要将第一行设置为“完形填空文本”和“背面附加信息”。
- 文件将被导入 Anki。你应该将每张卡片放在新的一行，并使用竖线分隔符 | 将完形填空文本和背面的附加信息分隔开。你应该返回一个 .txt 文件供我下载。
- 编写数学运算时，请使用 \( ... \) 标签包裹所有数学运算，例如 \( a^2+b^2=c^2 \) ]。默认情况下，这是内联数学运算。对于块数学运算，请使用 \[ ... \]。请在设置每张卡片的格式时自行决定。
- 书写化学方程式时，请使用以下格式 \( \ce{C6H12O6 + 6O2 -> 6H2O + 6CO2} \)，其中 \ce 是 MathJax 化学计算中必须使用的符号。
- 将所有内容放在代码块中。
- 请勿在答案或问题中为了视觉效果而使用换行符，因为换行符是新抽认卡的指示符。如果您需要列出所有内容，请使用 <br>。
