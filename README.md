# 个人知识库 (Personal Knowledge Base)

欢迎来到我的个人知识库！这是一个不断成长和完善的数字笔记集合，旨在记录、整理和分享我在学习、工作和兴趣探索过程中的知识、思考与经验。

## 📚 知识库概览

本知识库采用 Markdown 格式组织，并主要通过 [Obsidian](https://obsidian.md/) 进行管理和维护，以实现知识的互联互通和高效检索。

## 🎯 目的与理念

*   **学习沉淀**：记录和总结各领域的学习笔记，形成系统化的知识体系。
*   **问题解决**：积累项目经验、技术方案和故障排除过程，作为未来参考。
*   **思维整理**：捕捉灵感、整理思路，促进深度思考和知识创造。
*   **原子化笔记**：遵循“文件名即标题”的原则，力求每篇笔记聚焦一个核心概念或主题，提高复用性和可链接性。

## 📂 主要内容领域

本知识库涵盖以下主要领域：

*   **嵌入式系统与 ARM 架构**：深入探讨 ARM 处理器、寄存器、汇编、FreeRTOS 实时操作系统、STM32 开发、K210 等。
*   **计算机科学基础**：包括 CSAPP 相关内容、数据结构、编程语言 (C/C++/Python/Verilog)、Linux 操作系统、Git 等。
*   **人工智能 (AI)**：AI 学习路径、Prompt 工程、大模型应用等。
*   **个人成长与效率**：职业发展、认证考试、效率方法 (如 PARA 方法) 等。
*   **兴趣探索**：游戏、信息安全等。
*   **其他**：经济学、通用技术概念等。

## 结构与导航

知识库内容按主题和项目进行分类，主要目录结构如下：

*   `00_Inbox/`：临时存放新收集的资料和待整理的笔记。
*   `01_P/` (Projects)：与具体项目相关的笔记和资料。
*   `02_A/` (Areas)：长期关注的领域，如 C 语言、FreeRTOS 等。
*   `03_R/` (Resources)：各类学习资源、工具、Prompt 模板等。
*   `04_A/` (Archive)：已完成或不再活跃的项目/领域的归档。
*   `99_Definition/`：常用术语和概念的定义。
*   `99_Templates/`：笔记模板。
*   `attachments/`：存放笔记中引用的图片、PDF 等附件。

## 🛠️ 使用工具

*   **Obsidian**：核心笔记管理工具。
*   **Markdown**：笔记内容格式。
*   **Git**：版本控制，用于同步和备份知识库。
